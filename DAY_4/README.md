# 🦠 Day 4: Antimicrobial Resistance Surveillance
**Genomic Approaches for AMR | March 19, 2026**

## 🎯 Today's Objectives
Tackling one of the biggest challenges in One Health: Antimicrobial Resistance (AMR). We will use dedicated databases and machine learning to identify resistance profiles in genomic and metagenomic data.

## 📂 Notebooks in this Directory
* `Day4_AMR_Detection.ipynb`: Exploring AMR databases (ARKbase), identifying resistance genes/mutations, and characterizing resistance profiles.
* `Day4_MarkerML.ipynb`: Identifying marker features in metagenomic datasets using interpretable Machine Learning.

## 🛠️ Tools Covered
* **ARKbase 1.0**: AMR Knowledgebase.
* **MarkerML**: Machine learning for microbiome analysis.
