# 📊 Day 3: Comparative Genomic Analysis
**Deciphering Genetic Relationships | March 18, 2026**

## 🎯 Today's Objectives
We'll explore how pathogens evolve and differ by mapping variants, building phylogenetic trees, and performing advanced functional enrichment and structural analyses.

## 📂 Notebooks in this Directory
* `Day3_Variant_Calling.ipynb`: Mapping sequencing reads and identifying genomic mutations/SNPs.
* `Day3_Phylogenetics.ipynb`: Multiple sequence alignment and Maximum Likelihood-based tree construction.
* `Day3_getEnrich_and_Chitra.ipynb`: **Spotlight Session!** We will dive into pathway enrichment analysis using *getEnrich* and perform chromosomal rearrangement visualizations with *CHITRA*. 
* `Day3_croSSRoad.ipynb`: Running the high-throughput pipeline for SSR-driven comparative genomics.

## 🛠️ Tools Covered
* **getENRICH**: Gene/pathway enrichment analysis for non-model organisms.
* **CHITRA**: Chromosomal synteny and rearrangement analysis.
* **croSSRoad**: SSR analysis.
