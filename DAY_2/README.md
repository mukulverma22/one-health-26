# 🧬 Day 2: The Assembly Quest
**From Raw Data to Complete Genomes | March 17, 2026**

## 🎯 Today's Objectives
Today we move from raw sequencing reads to assembled genomes. We will learn how to clean up raw data, assemble viral genomes, and evaluate the quality of those assemblies.

## 📂 Notebooks in this Directory
* `Day2_QC_and_Trimming.ipynb`: Quality assessment and visualization of raw sequencing data using FastQC, Fastp, and MultiQC.
* `Day2_Assembly_Spades.ipynb`: Hands-on viral genome assembly using the Setu pipeline and SPAdes.
* `Day2_Quality_Assessment.ipynb`: Checking the completeness and detecting contamination in our new assemblies.

## 🛠️ Tools Covered
* **FastQC / Fastp / MultiQC**: For read quality control.
* **SPAdes / Setu**: For *de novo* assembly.
* **QUAST / BUSCO**: For assembly validation.
